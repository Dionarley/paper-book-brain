Basic Markdown Syntax
Element 	Markdown Syntax
Headings	# H1, ## H2, ### H3 (up to six #)
Bold	**bold text** or __bold text__
Italic	*italicized text* or _italicized text_
Bold & Italic	***bold italics*** or ___bold italics___
Strikethrough	~~The world is flat.~~
Blockquote	> blockquote
Ordered List	1. First item, 2. Second item, etc.
Unordered List	- Item or * Item or + Item
Code (inline)	`code`
Code Block	python print("Hello, World!")
Link	[Link text](https://www.example.com)
Image	![alt text](image.jpg)
Horizontal Rule	--- or *** or ___
